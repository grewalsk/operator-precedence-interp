## Headline (the honest, defensible claim)

> In Llama-3.1-8B (base **and** instruction-tuned), the operand **B is near-perfectly linearly
> decodable (CV-R² ≈ 0.99) from the post-bracket `)` residual of `(0+B)*C`, identically with and
> without few-shot context.** Yet the model **fails to compose zero-shot** (Instruct acc **0.265**,
> base 0.507) and **recovers to the inside-bracket ceiling with 2–4 in-context examples** (Instruct
> **0.92**). Because the operand's *encoding* is unchanged across regimes, the few-shot effect acts on
> downstream **use, not representation** — a **decodability/causal-use dissociation** in
> production-model arithmetic. The failure is **operation-specific** (addition composes), **depth-**
> and **surface-sensitive**, **magnitude-controlled**, and **unchanged by instruction tuning**.

This is the strongest version the evidence supports. The earlier "operand computed then *causally
discarded*" headline is **not** claimed here: the direct causal patch is inconclusive (see §5), so the
use-vs-encoding conclusion rests on a dissociation argument, not a direct ablation.

## 3. Brittleness characterization (Branch-B controls + 95% CIs)

**Operation-specific** (the key selectivity result): addition composes, multiplication does not.
- A1 `( 0 + B ) + C =` = **0.995**, A2 `0 + ( B + C ) =` = 0.9275, vs C1 (mult) = 0.265.
- Paired bootstrap: **Δ(A1−C1) = 0.730, 95% CI (0.688, 0.773)** [instruct].

**Magnitude-controlled** — C1 ≪ C4 at *every* matched |B·C| bin (instruct), so it is not "products got
bigger":

| \|B·C\| bin | n | acc(C1) | acc(C4) |
|-----------|---|---------|---------|
| [440,813)   | 79  | 0.532 | 0.962 |
| [813,1185)  | 126 | 0.278 | 0.944 |
| [1185,1558) | 104 | 0.163 | 0.885 |
| [1558,1930) | 64  | 0.188 | 0.891 |
| [1930,2303) | 27  | 0.000 | 0.704 |

**Surface contrast, CI-backed:** Δ(C4−C1) = **0.642, CI (0.595, 0.690)** [instruct]; **0.383, CI
(0.333, 0.433)** [base]. **Depth-sensitive:** D1 `( ( 0 + B ) ) * C =` (redundant nesting, same parse
as C1) = **0.0375** — one extra paren layer crashes it far below C1.

## 4. Few-shot robustness control (the reframing result) → FEWSHOT_FIXES_C1

| tag | 0-shot C1 | 2-shot | 4-shot | C4 ceiling |
|-----|-----------|--------|--------|------------|
| base     | 0.507 | 0.838 | 0.882 | 0.890 |
| instruct | **0.265** | 0.885 | **0.915** | 0.9075 |

Demos are per-item, distinct (B,C) pairs excluding the test pair (no answer leakage). **2–4 worked
examples of the same surface lift C1 to the C4 ceiling** → the zero-shot collapse is an
**elicitation/format fragility, not a binding-capability failure.** This is what retires the strong
"cannot bind the operands" claim and motivates WO#3 (does few-shot change the *encoding* or only the
*use*?).

**What C1 emits instead** (wrong-output distribution): overwhelmingly "other" (wrong products), not
systematic operand-grabbing — instruct: correct 0.265, **other 0.685**, equals_B 0.025, equals_C 0.025,
equals_B+C 0.000; base: correct 0.507, **other 0.460**, equals_B 0.015, equals_C 0.018.

## 5. Direct causal patch (salvage) — INCONCLUSIVE (reported honestly)

The redesigned operand-corruption salvage (corrupt B→B′, patch the clean `)` residual into the
corrupted run, with a site-matched C6 positive control) returned **`n_used = 0` on both tags →
INCONCLUSIVE.** Cause: the model's zero-shot C1 first-token output is **near-invariant to B**
(`n_no_contrast`: instruct 182, base 197 of the sampled C1-wrong pairs), so the contrast-based patch had
no first-token contrast to denoise. That B-invariance is itself *corroborating behavioral* evidence that
B is unused in the failing regime, but it is **not** a clean causal-ablation result. **We therefore do
not assert a direct causal "discarded" claim;** the use-vs-encoding conclusion is established by
elimination (§6).

*(Note: this run's earlier C6→C1-transplant design was a no-op by construction — identical `( 0 + B )`
prefix ⇒ identical `)` residual under causal masking — and was replaced; see `docs/wo2_redesign_note.md`.)*

## 6. WO#3 — few-shot decodability dissociation (the centerpiece)

Probe: is B linearly decodable (same dual-ridge CV-R², folds=5, ridge=1.0) from the **test
expression's** `)` residual under shots ∈ {0,2,4}? The site is the **LAST** `)` (the first closes a
shot's bracket); the `( 0 + B )` site assertion passed for **all 400 pairs at every shot level
(`n_skipped = 0`)**.

| tag | 0-shot R² | 2-shot | 4-shot | best layer | n_used | trend |
|-----|-----------|--------|--------|-----------|--------|-------|
| base     | 0.999 | 0.995 | 0.995 | 3→1→1 | 400 | **DECODABLE_IN_BOTH** |
| instruct | 0.999 | 0.994 | 0.994 | 3→1→1 | 400 | **DECODABLE_IN_BOTH** |

**B's encoding at the `)` site is near-perfect and unchanged across the failing (0-shot) and succeeding
(few-shot) regimes** (Δ ≤ 0.005, far inside the 0.05 stable tolerance), in both models. Combined with §4
(behavior swings 0.27→0.92), this isolates the few-shot effect to **downstream use, not representation.**

*Caveat:* R²≈0.99 at layers 1–3 is partly expected — B is a recent input token, so an early-residual
linear probe recovers it readily. The load-bearing point is the **dissociation** (decodable, identical
across regimes, while behavior swings), not the decodability magnitude per se.

## Interpretation / framing (locked)

Multiplying a bracketed operand is **brittle, operation-specific, depth-sensitive, surface-sensitive,
and magnitude-controlled** across base and instruction-tuned Llama-3.1-8B, **but recoverable with 2–4
in-context examples**, and the operand is **decodable identically with and without that context** — so
few-shot acts on **use, not encoding**. This is a clean **decodability/causal-use dissociation** in real
arithmetic on a production model. It does **not** include a direct causal-ablation result (salvage
inconclusive); it sits on the decodability + behavioral-dissociation line (Hewitt & Liang 2019 probing
controls; Sharma–Dawes–Raval 2026 decodability-vs-causal-use; Nikankin 2024 bag-of-heuristics).
