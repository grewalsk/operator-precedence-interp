## Step 5b — Branch-B controls (§9.B) + C6→C1 salvage (§10.B)

**Selectivity (operation-specific):**

| cond | surface | acc |
|------|---------|-----|
| A1 | `( 0 + B ) + C =` | **0.995** |
| A2 | `0 + ( B + C ) =` | **0.9275** |
| D1 | `( ( 0 + B ) ) * C =` (same parse as C1) | **0.0375** |

- **Addition composes** (A1/A2 ≈ 0.93–0.99) while multiplication-compose fails
  (C1=0.27) → **operation-specific**. *(Caveat: addition answers are small, so a
  magnitude-matched addition control would harden this.)*
- **D1=0.037** — one *redundant* paren layer (same value, same parse) crashes
  accuracy far below C1 → **depth-sensitive**; reconnects to the project's original
  nesting-depth question.

**Salvage — decodable-but-causally-unused (the causal centerpiece):**
- B linearly decodable from C1's post-bracket `)` site: **CV-R² = 0.957 @ layer 2**
  (n=33 used, 95 excluded as first-token-already-correct, 0 alignment skips).
- C6→C1 patch @ layer 2: **argmax-flip-to-correct rate = 0.00**, mean correct-logit
  delta = 0.00.
- **Reading: DECODABLE-BUT-UNUSED** — B is right there in the activations yet
  injecting the correctly-evaluated subexpression never makes C1 emit the correct
  product. The operand is computed and discarded.
- **By-layer decodability (the strong-claim test):** B is decodable at **R² ≈
  0.74–0.96 across ALL 32 layers** — 0.91 @ layer 0, peak 0.96 @ layer 2, and
  still **~0.80 at the mid-late layers (20–30)** where the outer `*C` should
  consume it (0.74 even @ the final layer 31). So B is present *where it matters*,
  not only trivially-early → the decodable-but-unused claim holds at full strength.
  *(Caveat: the causal patch was applied at the best-decode layer (2); a patch
  sweep across mid-late layers would further harden the "never used" half.)*

## Interpretation / framing (locked)

Multiplying a bracketed operand is **brittle, operation-specific, depth-sensitive,
and the operand is decodable-but-causally-discarded** — across both base and
instruction-tuned Llama-3.1-8B. Hits the decodability-vs-causal-use line
(Sharma–Dawes–Raval 2026) and Nikankin 2024 bag-of-heuristics.
